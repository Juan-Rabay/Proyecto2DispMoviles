Este proyecto es una aplicación móvil llamada Coffee App, diseñada para los amantes del café. Su objetivo es facilitar el almacenamiento, la gestión y la exploración de recetas de café, permitiendo a los usuarios:

Explorar recetas de café ya existentes, cada una con sus ingredientes, pasos detallados e imágenes.
Crear sus propias recetas de café y personalizarlas con ingredientes, instrucciones y fotos.
Guardar sus recetas favoritas para un acceso rápido en futuras preparaciones.
Compartir recetas fácilmente a través de otras aplicaciones, para que otros también puedan disfrutar de sus preparaciones.
Registrar estadísticas, como el número de veces que se ha preparado cada receta.

requerimientos claves: 

Almacenar todas las recetas en una base de datos local (SQLite) para un acceso rápido y sin conexión.
Permitir la carga inicial de recetas desde un archivo JSON, incluyendo los ingredientes, pasos, y fotos de referencia.
Usar el paquete share_plus para compartir recetas con amigos o en redes sociales, incluyendo el nombre, ingredientes y pasos de la receta.
Las recetas deben enviarse en un formato agradable para quien las reciba.
Implementar una lógica que permita mostrar imágenes tanto desde archivos locales como desde URL.
Si una imagen no está disponible, debe mostrarse un mensaje alternativo.
